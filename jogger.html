How to use this memory jogger

Instructions on how to use this memory jogger app:

  • Create a room entry for each space in your home.
  • Choose the common essentials to add to each room. (You will go more in depth when meeting with your specialist)
  • There are help buttons for each section if you need additional guidance!

Scroll down to begin!

How to add your rooms

  • First, select a room type using the dropdown on the left.
  • Next, you have the option of naming your room if the preset doesn't describe it properly.
  • Use short, simple names. You can add a custom room name in the optional field.
  • When you're all done, click the "Add room" button and you'll see it populate in the next section!

How to add your items

  • After you have added a room, click "choose items" to add towards that room!
  • You can select several items at once, then add them to the room with the add button.
  • You can remove items later if you change your mind.
